# Roi-Kham — Play

Public, playable build of the **Roi-Kham** Thai word quiz (game files only:
`index.html` + `words.js`). Served via GitHub Pages.

▶ **Play:** https://nonsfix.github.io/roi-kham-play/

This is a mirror of the game from the main (private) monorepo
[`NONSFIX/roi-kham`](https://github.com/NONSFIX/roi-kham) → `quiz-app/`. Source,
admin tools, and data pipeline live there.
