เรียนรู้ · ร้อยคำ

ไตรยางศ์

อักษรกลาง สูง ต่ำ และการผันวรรณยุกต์

กฎการสร้างคำ: อักษรกลาง (Mid-Class Consonants)

อักษรกลางในภาษาไทยมีทั้งหมด 9 ตัว คือ ก จ ด ต ฎ ฏ บ ป อ

(ท่องจำง่ายๆ ว่า: ไก่ จิก เด็ก ตาย(ด) เด็ก ตาย(ต) บน ปาก โอ่ง)

ลักษณะเด่นของคำที่มีพยัญชนะต้นเป็นอักษรกลาง:

  1. ผันวรรณยุกต์ได้ครบ 5 เสียง: เป็นกลุ่มเดียวที่สามารถผันเสียงได้ครบทั้ง สามัญ เอก โท ตรี และจัตวา (เมื่อเป็นคำเป็น)
  2. รูปวรรณยุกต์ตรงกับเสียง: เมื่อใส่วรรณยุกต์รูปใด ก็จะออกเสียงตามวรรณยุกต์นั้นตรงๆ ไม่มีการเลื่อนเสียงเหมือนอักษรต่ำ
  3. พื้นเสียง: หากไม่มีรูปวรรณยุกต์กำกับ (และเป็นคำเป็น) จะมีพื้นเสียงเป็น เสียงสามัญ

ตัวอย่างคำที่มีพยัญชนะต้นเป็นอักษรกลาง

1. คำว่า "บ้าน" (บ - า - น - ไม้โท)

  • พยัญชนะต้น: บ (อักษรกลาง)
  • วิเคราะห์: คำนี้ใช้วรรณยุกต์โท ( ้ ) และออกเสียงเป็นเสียงโทตรงตามรูป เพราะอักษรกลางผันได้ตรงตัว

2. คำว่า "ดาว" (ด - า - ว)

  • พยัญชนะต้น: ด (อักษรกลาง)
  • วิเคราะห์: คำนี้ไม่มีรูปวรรณยุกต์ และเป็นคำเป็น (แม่เกอว) พื้นเสียงจึงเป็น เสียงสามัญ โดยอัตโนมัติ

3. คำว่า "โก๊ะ" (ก - โ - ะ - ไม้ตรี)

  • พยัญชนะต้น: ก (อักษรกลาง)
  • วิเคราะห์: คำนี้ใช้วรรณยุกต์ตรี ( ๊ ) และออกเสียงเป็นเสียงตรีตรงตามรูป (ซึ่งอักษรต่ำหรืออักษรสูงจะใช้ไม้ตรีผันตรงตัวแบบนี้ไม่ได้)

กฎการสร้างคำ: อักษรสูง (High-Class Consonants)

อักษรสูงในภาษาไทยมีทั้งหมด 11 ตัว คือ ข ฃ ฉ ฐ ถ ผ ฝ ศ ษ ส ห

(หลักการจำยอดนิยม: ผี ฝาก ถุง ข้าว สาร ให้ ฉัน + ฃ ฅ ฐ ฤ ฤๅ)

ลักษณะเด่นของคำที่มีพยัญชนะต้นเป็นอักษรสูง:

  1. พื้นเสียงเป็นเสียงจัตวา: หากเป็นคำเป็นและไม่มีรูปวรรณยุกต์กำกับ จะออกเสียงเป็น เสียงจัตวา (เสียงสูง) โดยอัตโนมัติ (ต่างจากอักษรกลาง/ต่ำ ที่พื้นเสียงเป็นสามัญ)
  2. ผันได้ 3 เสียง: คือ เสียงเอก, เสียงโท, และเสียงจัตวา
  3. รูปกับเสียงตรงกัน:
    • ใส่ไม้เอก ( ่ ) = ออกเสียงเอก
    • ใส่ไม้โท ( ้ ) = ออกเสียงโท

ตัวอย่างคำที่มีพยัญชนะต้นเป็นอักษรสูง

1. คำว่า "สวย" (ส - ว - ย)

  • พยัญชนะต้น: ส (อักษรสูง)
  • วิเคราะห์: คำนี้ไม่มีรูปวรรณยุกต์ แต่เนื่องจาก "ส" เป็นอักษรสูง พื้นเสียงจึงเป็น เสียงจัตวา (เสียงเหมือนมีไม้จัตวากำกับ แต่อยู่ในรูปปกติ)

2. คำว่า "ไข่" (ข - ไ - ไม้เอก)

  • พยัญชนะต้น: ข (อักษรสูง)
  • วิเคราะห์: คำนี้ใช้รูปไม้เอก ( ่ ) และออกเสียงเป็น เสียงเอก ตรงตามรูป (ข + ไ + ่ = ไข่)

3. คำว่า "เสื้อ" (ส - เ - ื - อ - ไม้โท)

  • พยัญชนะต้น: ส (อักษรสูง)
  • วิเคราะห์: คำนี้ใช้รูปไม้โท ( ้ ) และออกเสียงเป็น เสียงโท ตรงตามรูป (ส + เ-ือ + ้ = เสื้อ)

ข้อสังเกต: อักษรสูงจะไม่ใช้รูปไม้ตรี ( ๊ ) หรือไม้จัตวา ( ๋ ) ในการเขียนคำทั่วไป เพราะพื้นเสียงเป็นจัตวาอยู่แล้ว และไม่สามารถผันเสียงสามัญหรือเสียงตรีได้ด้วยตัวเองครับ

กฎการสร้างคำ: อักษรต่ำ (Low-Class Consonants)

อักษรต่ำมีจำนวนมากที่สุดในภาษาไทย คือ 24 ตัว แบ่งเป็น

  • อักษรต่ำคู่ (คู่กับอักษรสูง): พ ภ, ค ฅ ฆ, ฟ, ฑ ฒ ท ธ, ซ, ช ฌ, ฮ
  • อักษรต่ำเดี่ยว (ไม่มีคู่): ง, ญ, น, ย, ณ, ร, ว, ม, ฬ, ล

กฎสำคัญที่ต้องระวัง (ความแตกต่างจากอักษรกลางและสูง):

  1. รูปวรรณยุกต์ไม่ตรงกับเสียง: เป็นกลุ่มเดียวที่ "รูป" กับ "เสียง" เลื่อนกัน 1 ขั้นเสมอ
    • ใส่ ไม้เอก ( -่ ) → ออก เสียงโท
    • ใส่ ไม้โท ( -้ ) → ออก เสียงตรี
  2. พื้นเสียง: หากเป็นคำเป็นและไม่มีรูปวรรณยุกต์ จะเป็น เสียงสามัญ
  3. ข้อจำกัด: อักษรต่ำไม่สามารถผันเสียง เอก หรือ จัตวา ได้ตามลำพัง (ต้องใช้ ห นำ หรือ อ นำ มาช่วย)

ตัวอย่างคำที่มีพยัญชนะต้นเป็นอักษรต่ำ

1. คำว่า "นา" (น - า)

  • พยัญชนะต้น: น (อักษรต่ำ)
  • วิเคราะห์: คำนี้ไม่มีรูปวรรณยุกต์ และเป็นคำเป็น พื้นเสียงจึงเป็น เสียงสามัญ (ราบเรียบ)

2. คำว่า "พ่อ" (พ - อ - ไม้เอก)

  • พยัญชนะต้น: พ (อักษรต่ำ)
  • วิเคราะห์: คำนี้ใช้ รูปไม้เอก ( -่ ) แต่ออกเสียงเป็น เสียงโท (เสียงตกลง)
  • กฎ: อักษรต่ำ + ไม้เอก = เสียงโท

3. คำว่า "น้ำ" (น - ำ - ไม้โท)

  • พยัญชนะต้น: น (อักษรต่ำ)
  • วิเคราะห์: คำนี้ใช้ รูปไม้โท ( -้ ) แต่ออกเสียงเป็น เสียงตรี (เสียงสูง)
  • กฎ: อักษรต่ำ + ไม้โท = เสียงตรี

สรุปภาพรวมไตรยางศ์ (อักษร 3 หมู่):

  • กลาง: ผันครบ 5 เสียง / รูปตรงเสียง
  • สูง: พื้นเสียงจัตวา / ผันได้ 3 เสียง / รูปตรงเสียง
  • ต่ำ: พื้นเสียงสามัญ / ผันได้ 3 เสียง / รูปไม่ตรงเสียง (เอกเป็นโท, โทเป็นตรี)
